### API 端点
- `POST /api/training/run` - 训练模型（接收数据配置、训练参数、模型结构）
- `GET /api/training/list` - 训练记录列表
- `GET /api/training/{id}` - 训练结果详情（含 epoch 日志）
- `DELETE /api/training/{id}` - 删除训练记录和模型文件
- `WS /api/training/{id}/ws` - WebSocket 实时训练进度推送

## WebSocket 实时训练监控改造

将训练从同步阻塞改为异步后台执行 + WebSocket 实时推送。

### 改动要点
- `POST /run` 改为立即返回 `{ id, status: "training" }`，训练在后台线程执行
- 新增 `TrainingEventBus` 事件总线：Keras callback → asyncio.Queue → WebSocket
- WS 鉴权通过 query param `?token=xxx`
- 30s 心跳防连接超时
- Pydantic V2 保留字段问题：`model_path` → `saved_model_path`，`model_config` → `network_config`
- Keras 3.x 兼容：`fit_generator` → `fit()`
- 运行环境：FastAPI 跑在 Python 3.10，需在此环境安装 tensorflow
